# Django Rest Api

> Django REST framework boilerplate with JWT and Swagger

![CI](https://img.shields.io/badge/status-active-brightgreen)
![License](https://img.shields.io/badge/license-MIT-blue)

## Tech Stack

Django, DRF, PostgreSQL

## Getting Started

````bash
git clone https://github.com/your-org/django-rest-api.git
cd django-rest-api
npm install
npm run dev
````
